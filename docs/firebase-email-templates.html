Firebase Console Setup Instructions

Firebase Authentication supports one custom template per language for each template type. Here's how to set them up:

  1. Go to Firebase Console → Authentication → Templates
  2. Click the pencil icon on the template you want to customize (e.g., "Email address verification")
  3. Set the default language template (German/de) by editing the template body
  4. Click "Customize by language" to add English (en) and Spanish (es) versions
  5. For each language, paste the corresponding HTML from this preview
  6. Set the Subject line for each language (see below)

How Language Switching Works (Code)

In AuthContext.tsx, the app sets auth.languageCode = getLocale() before calling sendEmailVerification() or sendPasswordResetEmail(). This tells Firebase which language template to use. The locale is determined by the language the user selected via the LanguageSwitcher component on the login/signup screen.
