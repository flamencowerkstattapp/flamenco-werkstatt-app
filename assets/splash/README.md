# PWA Splash Screens Guide

This directory should contain iOS splash screens for the PWA.

## Required Splash Screen Sizes

Create splash screens with the app branding (logo on #8B0000 background):

- **splash-2048x2732.png** - iPad Pro 12.9" (portrait)
- **splash-1668x2388.png** - iPad Pro 11" (portrait)
- **splash-1536x2048.png** - iPad Pro 9.7" (portrait)
- **splash-1125x2436.png** - iPhone X/XS/11 Pro (portrait)
- **splash-1242x2688.png** - iPhone XS Max/11 Pro Max (portrait)
- **splash-828x1792.png** - iPhone XR/11 (portrait)
- **splash-1242x2208.png** - iPhone 8 Plus (portrait)
- **splash-750x1334.png** - iPhone 8/7/6s (portrait)
- **splash-640x1136.png** - iPhone SE (portrait)

## Design Guidelines

Each splash screen should:
- Have a background color of #8B0000 (dark red)
- Display the app logo centered
- Match the app's branding

## How to Generate

### Option 1: Online Tool
Use https://www.appicon.co/ or https://apetools.webprofusion.com/app/#/tools/imagegorilla

### Option 2: Manual Creation
Use design software (Photoshop, Figma, Canva) to create each size with:
- Background: #8B0000
- Centered logo from ../logo.png
- Safe area margins (avoid edges)
